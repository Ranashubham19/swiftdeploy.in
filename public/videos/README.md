# Video Assets for SwiftDeploy AI

## Required Video Files

Place the following video files in this directory:

### Main Tutorial Video
- `telegram-token-tutorial.mp4` - Primary video format (H.264 codec, 1080p recommended)
- `telegram-token-tutorial.webm` - WebM format for better browser compatibility
- `discord-demo.mp4` - Discord setup tutorial reel (H.264 codec)
- `discord-demo.webm` - Optional WebM fallback for Discord tutorial

## Video Content Requirements

The video should demonstrate step-by-step how to:
1. Open Telegram app
2. Search for @BotFather
3. Start a chat with BotFather
4. Type /newbot command
5. Name your bot
6. Choose a username
7. Receive and copy the bot token

## Recommended Specifications
- Duration: 30-60 seconds
- Resolution: 1080x1920 (portrait) or 1080x608 (landscape)
- Format: MP4 (H.264) and WebM (VP9)
- File size: Under 10MB for optimal loading
- Loop: Should be designed to loop seamlessly

## Fallback Images

If videos aren't available, the system will look for:
- `public/images/telegram-tutorial-poster.jpg` - Loading poster image
- `public/images/telegram-tutorial-steps.jpg` - Static fallback image

## Creation Tips

You can create this video using:
- Screen recording software (OBS, CamStudio, etc.)
- Mobile screen recording
- Video editing software (DaVinci Resolve, Adobe Premiere, etc.)
- AI video generation tools
- Simple step-by-step screenshots with animation

The video will automatically play in a loop on the Connect Telegram page.
